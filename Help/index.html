Introduction

traveling-salesman-problem, is a MacOS App written by David Nadoba in Swift to find the shortest path Implementation of different pathfinding algorithms by using a bidirectional and weighted graph data structure `WeightedGraph`.

You can find a generic implementation of such a graph struct in `WeightedGraph.swift` and its related protocols. This was my first macOS App and it uses only cocoa bindings in the `RouteSummaryViewController`.

He wrote this App for the Code Competition "Der Handlungsreisende from IT-Talents.de.

You can download MacOS App (Release v1.0) from github, and view its home page at traveling-salesman-proiblem.
