# aplicación macOS escrita en Swift para encontrar el camino más corto
Implementación de diferentes algoritmos de pathfinding mediante el uso de una estructura de datos de grafos bidireccional y ponderada `WeightedGraph`. Puede encontrar una implementación genérica de dicha estructura de gráfico en `WeightedGraph.swift` y sus protocolos relacionados.
Esta fue mi primera aplicación de macOS y solo utiliza enlaces de cacao en el `RouteSummaryViewController`.
Escribí esta aplicación para [Code Competition "Der Handlungsreisende"] (https://www.it-talents.de/foerderung/code-competition/code-competition-02-2017) de [IT-Talents.de] ( https://www.it-talents.de/).
